## string.capitalize

Apply uppercase to first letter of string

```javascript
import { capitalize } from '@utils/string/capitalize';
```

### Usage

```javascript
capitalize(value)
```

**Params:**

* **value**: string - string to be capitalized

**Return:**

* string - capitalized string

#### Example

```javascript
capitalize('hello, world!');
```

#### Result

```json
"Hello, world!"
```
