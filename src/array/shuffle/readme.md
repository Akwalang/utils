## array.shuffle

Shuffle array items

```javascript
import shuffle from '@utils/array/shuffle';
```

### Usage

```javascript
shuffle(array)
```

**Params:**

* **array**: T[] - array of values

**Return:**

* T[] - array with shuffled items

#### Example

```javascript
shuffle([1, 2, 3, 4, 5, 6]);
```

#### Result

```json
[4, 3, 5, 6, 1, 2]
```
